# Express + MongoDB

## Requirements

- Node.js v12.16.1
- MongoDB v4.2.3
- npm v6.13.4

## Installation

```bash
npm install
```

## Usage

### Run server
Installs dependencies and starts the server in development mode with nodemon.

```bash
npm run dev
```

### Run tests
Runs all tests in the `tests` directory with jest.

```bash
npm run test
```

### Run linter
Runs eslint on all files in the `src` directory.

```bash
npm run lint
```
